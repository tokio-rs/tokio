# Tokio FS

Asynchronous filesystem manipulation operations (and stdin, stdout, stderr).

[Documentation](https://tokio-rs.github.io/tokio/tokio_fs/)

## Overview

This crate provides filesystem manipulation facilities for usage with Tokio.
